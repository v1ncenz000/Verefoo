VEREFOO REST APIs docs

What Verefoo can do through the REST APIs

Verefoo furnishes a set of REST APIs, grouped by type of resource, intended to launch simulations on the defined resources. Here follows the list of services available:

  1. Store and retrieve Service Graphs and Allocation Graphs;
  2. Store and retrieve the constraints related to a Service Graph;
  3. Store and retrieve the Network Security Requirements to satisfy;
  4. Store and retrieve the Substrate Networks where VNFs can be deployed;
  5. Instruct which Network Security Functions can be allocated by the framework;
  6. Run simulations and retrieve the results.

How to use the REST APIs

For any information or doubt regarding the REST APIs, refer to the Swagger documentation.
